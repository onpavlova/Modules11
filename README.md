Приложение "Телефонный справочник"

Приложение работает с файлом следующей структуры:
ФИО_1; Телефон_1; Комментарий_1
ФИО_2; Телефон_2; Комментарий_2

Функционал:
открыть файл
сохранить файл
показать все контакты
создать контакт
найти контакт
изменить контакт
удалить контакт
выход

При реализации использован паттерн MVC.
Код разбит на 3 модуля view, controller, model.
Созданы классы для справочника, контакта, чтения файла и записи файла.
Созданы кастомные исключения для обработки всевозможных ошибок.
